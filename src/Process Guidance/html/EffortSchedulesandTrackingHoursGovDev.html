Effort, Schedules, and Tracking Hours (GovDev)
Send Feedback

You use the fields that are grouped under Effort (hours)to track estimated, completed, and remaining work for a Task and to report on the progress toward completing tasks. By tracking hours, you gain more detailed insight into which use casees are almost finished and what the team burn rate is. Several reports and dashboards provide charts that display the data that these fields contain.

You specify the level of effort for the Tasks that you are assigned. For more information about how to track hours, see Task (GovDev) .

Scheduling Fields That Office Project Calculates

The following table lists fields that are calculated if you create a project plan in Office Project and then synchronize that plan with Tasks that are stored in Team Foundation Server. These fields do not appear on the work item form, but they are calculated for both use cases and Tasks that are linked to use cases. You can view their read-only values in results from a query or from Office Excel and Office Project. For more information, see Scheduling Tasks and Assigning Resources Using Microsoft Project.
